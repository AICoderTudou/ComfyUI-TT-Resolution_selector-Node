# ComfyUI TT分辨率选择器插件

一个简单实用的 ComfyUI 插件，提供常用分辨率的下拉选择功能。

## 功能特性

- 🎯 提供20+种常用分辨率预设，并带有横屏/竖屏/方形备注
- 📐 包含多种宽高比：1:1, 2:3, 3:2, 4:7, 7:4, 16:9, 9:16等
- �️ 支持自定义分辨率输入功能
- �🔄 返回标准的width和height整数值
- 🎨 适用于图像生成、处理等场景

## 安装方法

### 方法1：直接下载
1. 下载本插件的所有文件
2. 将整个文件夹复制到 `ComfyUI/custom_nodes/` 目录下
3. 重启 ComfyUI

### 方法2：Git克隆
```bash
cd ComfyUI/custom_nodes/
git clone https://github.com/yourusername/comfyui-resolution-selector.git
```

## 使用方法

1. 在 ComfyUI 中右键添加节点
2. 选择 `utils` -> `TT分辨率选择器`
3. **预设模式**：直接在下拉菜单中选择需要的分辨率
4. **自定义模式**：
   - 打开 `use_custom_resolution` 开关
   - 在 `custom_width` 和 `custom_height` 中输入自定义数值
   - 此时下拉选择的分辨率将被忽略
5. 节点会输出对应的 width 和 height 值
6. 将输出连接到需要分辨率参数的其他节点

## 输出说明

- **width**: 图像宽度（整数）
- **height**: 图像高度（整数）
