# ComfyUI TT分辨率选择器插件

一个简单实用的 ComfyUI 插件，提供常用分辨率的下拉选择功能。

## 功能特性

- 🎯 提供20+种常用分辨率预设
- 📐 包含多种宽高比：1:1, 2:3, 3:2, 4:7, 7:4, 16:9, 9:16等
- 🔄 返回标准的width和height整数值
- 🎨 适用于图像生成、处理等场景

## 支持的分辨率

### 正方形 (1:1)
- 512x512
- 768x768  
- 1024x1024

### 竖屏比例
- 512x768 (2:3)
- 512x896 (4:7)
- 640x960 (2:3)
- 832x1216 (13:19)
- 1024x1536 (2:3)
- 720x1280 (9:16)
- 1080x1920 (9:16)
- 768x1344 (4:7)
- 896x1152 (7:9)

### 横屏比例
- 768x512 (3:2)
- 896x512 (7:4)
- 960x640 (3:2)
- 1216x832 (19:13)
- 1536x1024 (3:2)
- 1280x720 (16:9)
- 1920x1080 (16:9)
- 1344x768 (7:4)
- 1152x896 (9:7)

## 安装方法

### 方法1：直接下载
1. 下载本插件的所有文件
2. 将整个文件夹复制到 `ComfyUI/custom_nodes/` 目录下
3. 重启 ComfyUI

### 方法2：Git克隆
```bash
cd ComfyUI/custom_nodes/
git clone https://github.com/yourusername/comfyui-resolution-selector.git
```

## 使用方法

1. 在 ComfyUI 中右键添加节点
2. 选择 `utils` -> `TT分辨率选择器`
3. 在下拉菜单中选择需要的分辨率
4. 节点会输出对应的 width 和 height 值
5. 将输出连接到需要分辨率参数的其他节点

## 输出说明

- **width**: 图像宽度（整数）
- **height**: 图像高度（整数）
